# Flappy
Flappy is a 2D arcade game inspired by the popular Flappy Bird. The twist is that you control two birds at the same time, one with each arrow key. How long can you survive the obstacles?



Installation
To install and play Flappy, just clone repo and run the exe file

Gameplay
The goal of Flappy is to avoid hitting the pipes, sometimes the pipes are cloed. Pres "Q" to launch a missile and open the pipe!

The game features:

Simple but addictive gameplay
Retro pixel art graphics
High score system
Sound effects and music
Flappy was developed by Christio02 as his first game project
